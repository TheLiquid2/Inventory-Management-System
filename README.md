# Inventory Management System
Description:

An inventory management system developed during my software engineering internship.
The project focuses on managing inventory data through a backend built with Node.js
and database integration.

What Technologies that are used and gained during my internship:
- Node.js
- JavaScript
- Database
- HTML, CSS

Features of the system were:
- Add, update, and delete inventory items.
- Store and retrieve inventory data from a database.
- Backend logic implemented using Node.js.
- Frontend interface for interacting with the system.

What I Learned:
- Building backend logic using Node.js
- Handling requests and responses
- Connecting Node.js applications to a database
- Implementing CRUD operations
- Understanding real-world system workflows during internship
